# Repository Agent Notes

## README Maintenance

- In `README.md`, keep dated entries under `Current Status / 当前状态` in reverse chronological order. Newer updates must appear above older updates in both the English and Chinese subsections.
- When adding a dated update, preserve the bilingual structure: add the English update and the matching Chinese update in the same relative position.

## Manuscript Voice

- Paper-facing prose, table notes, and figure notes must not contain author-facing workflow notes, internal discussion, or draft-management explanations. Keep those in task READMEs, revision memos, or author-facing summaries instead.
- When adding writing rules, distinguish reader-facing scope/identification explanations from author-memo language. Phrases such as "需要说明的是" are not automatically forbidden; they are acceptable only when they introduce substantive information the reader needs.
