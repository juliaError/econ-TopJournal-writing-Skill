# Main Regression Tables

## Admission Rule

A main regression table belongs in the main text only if it carries the paper's central estimate or the key comparison that supports the research question.

## Default Structure

- Put the focal coefficient or interaction first.
- If the focal term is an interaction, report lower-order terms that remain identifiable under the fixed effects.
- Show the dependent variable clearly in the header or panel title.
- Add controls and fixed effects progressively only when the progression teaches identification or interpretation.
- Put fixed effects, controls, clustering, observations, and fit statistics below the coefficient block.

## Column Count

- Preferred main-text vertical table: 3-6 substantive columns per panel.
- A 2-column table is usually too thin unless it is a compact theory or calibration result.
- A 7+ column table should be split, moved to landscape, or placed in the appendix unless the horizontal comparison is essential.

## Placement

- Main table should appear soon after empirical design and before robustness.
- Descriptive statistics may precede it only if the sample or variable construction is not obvious.
- Do not keep insignificant columns in the main table unless they are needed for identification or interpretation.

## Check

The table should answer: what is estimated, on whom, with what comparison, and how large the result is.
