# Column Count And Panel Rules

## Column Count

- Main vertical tables: usually 3-6 substantive columns per panel.
- Compact descriptive tables: 4-8 columns are acceptable if variables are rows.
- Wide horizontal tables: use only when 7+ columns are essential to comparison.
- Appendix tables can be wider, but must still fit the final PDF or Word page.

## Panel Design

- Use panels for different dependent variables, mechanism families, sample groups, or analysis levels.
- Panel titles should be standalone and meaningful.
- Within each panel, column numbering and header structure should remain consistent.
- Do not mix different analysis levels without saying so in the text and note.

## Width Fix Order

1. Remove vertical lines and unnecessary whitespace.
2. Shorten headers and use controlled line breaks.
3. Reduce low-information columns.
4. Split into panels or separate tables.
5. Adjust column widths and row spacing.
6. Only then consider mild scaling.

## Check

No table is acceptable if it overflows the page, has misaligned standard errors, or uses a different font/size system from the rest of the paper.
