# Event-Study Figures

## Rule

Event-study figures should reveal dynamics, pre-trends, and post-treatment timing. They should not be decorative versions of a regression table.

## Required Elements

- Event time on the x-axis.
- Coefficient or effect size on the y-axis.
- Confidence intervals or standard error bars.
- Baseline period clearly omitted or shown as zero.
- Vertical reference line at treatment or policy timing.
- Horizontal zero line.

## Design Rules

- If the baseline period is omitted from estimation, add a visible zero point at the baseline in the plotted series.
- Use one primary color for coefficients and a lighter version or thin line for confidence intervals.
- Keep gridlines minimal; zero and treatment reference lines matter more than background grids.
- Use the same y-axis scale across panels when comparing groups.

## What To Avoid

- Connecting event time around the omitted base period without making the base explicit.
- Changing color, y-scale, or confidence interval style across similar event-study figures.
- Plotting too many subgroup lines in one panel.

## Check

The reader should immediately see whether pre-trends are flat, when the effect begins, and how persistent it is.
