# English Diction Coverage Policy

## Principle

This file records how to think about coverage behind `english-diction` without publishing a paper list or source identifiers. Use it to avoid treating a single article's style as a universal rule.

Public skill files include abstracted rules and original examples only. They do not store source text, long quotations, paper names, or per-paper extraction logs.

## Coverage Categories

Use English diction rules only when they are supported by broad patterns across more than one kind of economics paper:

- theory and model prose;
- empirical identification prose;
- data and measurement prose;
- results and magnitude prose;
- literature and contribution prose;
- limitations, scope, and external-validity prose;
- table and figure narration.

## Downgraded Or Missing Source Handling

If a local source has weak text extraction, restricted access, or incomplete main text, do not use it for sentence-level rules. It may remain in private audit notes, but public rules should not name it or rely on it as sole support.

## Function Coverage

| Function | Coverage Standard | Use |
| --- | --- | --- |
| Research question and setup | Must work across theory and empirical introductions | Abstract openings, introduction question paragraphs |
| Model setup | Must not depend on one model's notation | Environments, players, timing, information |
| Assumption and equilibrium | Must translate formal objects into plain economic content | Assumptions, equilibrium definitions, propositions |
| Mechanism and intuition | Must explain behavior, constraint, margin, or channel | Intuition paragraphs and mechanism interpretation |
| Identification and variation | Must name comparison, variation, and threat | Empirical strategy and design summaries |
| Data and measurement | Must distinguish concept, measure, and sample | Data sections and constructed variables |
| Results and magnitude | Must state direction, size, and interpretation | Table discussion and abstract findings |
| Literature and contribution | Must name the margin added, not just importance | Contribution paragraphs and related literature |
| Limitation and scope | Must bound the claim without undermining it | External validity, caveats, interpretation boundaries |
| Table and figure narration | Must help the reader interpret evidence | Table notes, figure discussion, appendix references |

## Use Check

Before applying a rule, ask:

1. Is this a theory sentence, empirical sentence, or general prose sentence?
2. Does the rule reflect broad economics-writing practice rather than one source's idiosyncrasy?
3. Is the rewrite an original sentence rather than source imitation?
4. Does the final sentence make a narrower, clearer, and more credible claim?
