# 中文经济学常用搭配

## 使用原则

优先使用朴素、稳定、对象明确的搭配。少用看似有气势但不能增加信息的词。

## 研究动作

| 场景 | 推荐搭配 | 慎用搭配 | 检查标准 |
| --- | --- | --- | --- |
| 提出问题 | 考察、讨论、识别、刻画 | 深刻揭示、全面剖析 | 是否说明研究对象 |
| 测度变量 | 构造、度量、分解、校正 | 打造指标、赋能测度 | 是否说明口径和来源 |
| 理论机制 | 解释、约束、传导、改变 | 形成闭环、充分验证 | 是否说明机制链条 |
| 实证检验 | 估计、比较、检验、识别 | 跑出结果、验证一切 | 是否说明识别对象 |
| 政策含义 | 提供证据、表明、提示 | 提供重要启示、助力发展 | 是否有具体政策对象 |

## 结果动词

| 结果类型 | 推荐动词 | 示例骨架 |
| --- | --- | --- |
| 增加 | 提高、扩大、增强、促进 | `[X] 提高了 [Y]` |
| 减少 | 降低、缩小、削弱、抑制 | `[X] 降低了 [Y]` |
| 风险 | 加剧、缓解、转移、分散 | `[X] 缓解了 [主体] 面临的 [风险]` |
| 结构 | 改变、重塑、调整、分化 | `[X] 改变了 [主体] 的 [结构/配置]` |
| 机制 | 作用于、传导至、约束、激励 | `[X] 通过 [机制] 作用于 [Y]` |

## 常见名词搭配

| 名词 | 稳定搭配 | 避免写法 |
| --- | --- | --- |
| 机制 | 作用机制、传导机制、约束机制 | 机制闭环、完整链条闭环 |
| 识别 | 识别策略、识别前提、识别威胁 | 主识别任务、识别抓手 |
| 贡献 | 边际贡献、经验贡献、机制补充 | 重要贡献、创新性贡献 |
| 政策 | 政策工具、政策目标、政策约束 | 政策赋能、政策抓手 |
| 数据 | 数据来源、统计口径、匹配过程 | 数据基础扎实、数据支撑有力 |
| 结果 | 估计结果、主要结果、稳健性结果 | 显著结果、有意义结果 |

## 连接词节制

- 同一段中“这说明/这表明/这意味着”最多出现一次。
- 能用因果主语推进时，不用连接词硬接。
- “进一步地”只在真正递进时使用，不用于普通换段。
- “换言之”只用于重述复杂逻辑，不用于重复上一句。

## 更自然的替换

| 常见 AI 腔 | 更稳写法 |
| --- | --- |
| 深刻揭示了 X 的重要作用 | 表明 X 与 Y 存在稳定关系 |
| 为 X 提供重要启示 | 为理解/评估 X 提供经验证据 |
| 形成机制闭环 | 与 [机制] 的解释相一致 |
| 赋能企业发展 | 改善企业的 [融资/需求/生产率] |
| 助力高质量发展 | 影响 [具体对象] 的 [具体结果] |
| 具有重要现实意义 | 对 [具体政策问题] 具有参考价值 |

## 检查标准

每个动词都应能回答三个问题：

1. 谁在行动？
2. 作用于什么对象？
3. 改变了哪个可观察结果？
