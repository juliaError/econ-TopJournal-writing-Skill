# 中文遣词规则来源覆盖

## 使用原则

本文件说明 `chinese-diction` 规则库的语料覆盖范围。使用这些规则时，应把它们视为从多篇中文经济学论文中抽象出的句法、搭配和修稿规则，而不是任何单篇论文的原句模板。

公开 skill 不保存论文长原句。若需要回到原文核对，请使用本地任务目录中的审计文件，而不要在公开规则中加入长引文。

## 纳入状态

- 适用场景：判断某条中文遣词规则是否有足够语料支撑。
- 推荐写法：优先使用被多类论文共同支持的规则；对只来自少数文本的风格，写成可选倾向，而不是硬规则。
- 反面写法：把某一篇论文的表达偏好直接写成“顶刊通用写法”。
- 检查标准：规则能否说明其适用场景、推荐写法、反面写法和检查标准。

## 降级与缺失来源

C15 与 C18 当前不得作为自动规则的核心支撑。若用户后续提供可读文本或人工校正版本，可以把它们用于补充“指数构建、空间事实、企业投资行为”相关表达，但需要重新记录审计结果。

C10 与 C14 当前不纳入自动蒸馏。N02 与 C10 主题相近，但不是同一篇论文，不能用 N02 替代 C10 的来源记录。

## 使用检查

在应用中文遣词规则前，先问四个问题：

1. 当前句子属于哪一种功能：问题、概念、机制、识别、结果、文献、贡献、边界，还是表图说明？
2. 该功能是否有多篇来源共同支持，而不是来自单篇风格？
3. 当前改写是否只是借用抽象模式，而没有复制论文原句？
4. 改写后是否更像论文正文，而不是作者备忘、AI 总结或英文直译？
